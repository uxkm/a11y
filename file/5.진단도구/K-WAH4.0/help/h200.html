K-WAH 4.4 특징

• 한국형 웹 콘텐츠 접근성 지침 2.0 지원

한국형 웹 콘텐츠 접근성 지침 2.0에 의거하여 6개 세부지침에 대한 접근성 준수여부를 자동으로 점검합니다.

• 다이나믹 방식 주소 수집

K-WAH 4.4는 웹 페이지 랜더링에 의한 다이나믹 주소 수집으로 페이지 수집 기능을 강화하였습니다. 또한 소스 다운방식을 보조적인 수단으로 추가하여 주소 수집 기능을 보완하였습니다.

• 페이지 중복 수집 가능성 배제

하나의 웹페이지가 중복 수집되는 것을 방지하여 더 많은 페이지를 점검할 수 있습니다.

• 마크업 문법 검사

마크업 문법은 DTD에 따라 평가하도록 하므로써 문법 검사의 정확도를 높였습니다. 적용 가능한 DTD 규격은 다음과 같습니다.

DTD 유형 선언문
HTML4.01 Strict <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
HTML4.01 Transitional <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
HTML4.01 Frameset <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Frameset//EN" "http://www.w3.org/TR/html4/frameset.dtd">
XHTML1.0 Strict <!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml1-strict.dtd">
XHTML1.0 Transitional <!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-transitional.dtd">
XHTML1.0 Frameset <!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.0 Frameset//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-frameset.dtd">
XHTML1.1 <!DOCTYPE HTML PUBLIC "-//W3C//DTD XHTML 1.1//EN" "http://www.w3.org/TR/xhtml11/DTD/xhtml11.dtd">
HTML5 <!DOCTYPE html>

상기 표의 DTD유형 외에도 유사한 DTD규격(XHTML Basic 1.0, XHTML Basic 1.1 등)을 처리할 수 있습니다.


• 다양한 주소 유형 수집 가능

K-WAH 4.4는 K-WAH4.0과 달리 URL 점검시 확장자를 자동 판별할 수 있어서 확장자를 별도로 지정하지 않더라도 다양한 확장자를 사용하는 웹 페이지에 대한 평가가 가능합니다. 특히 히기능은 모바일 웹페이지의 평가에 적합합니다.

• 소프트웨어 접근성 준수

소프트웨어 접근성을 준수하여 모든 기능을 키보드로 사용 가능하도록 하였습니다.

• 인코딩 적용 설정

K-WAH4.4은 인코딩을 선언하지 않은 웹 소스의 인코딩 방법을 설정할 수 있도록 함으로써 웹소스 추출시에 글자가 깨지는 문제를 해결할 수 있습니다.

• 사용성 증진

내장된 웹브라우저로부터 URL 직접 입력이 가능하며 점검 진행 상황을 자세히 표시하여 프로그램의 사용성을 향상시켰습니다.

• JavaScript 주소 수집

K-WAH4.4은 JavaScript 주소 수집이 가능하므로 더 많은 유형의 웹 사이트에 접근하여 접근성 평가가 가능합니다.

• 보안 강화 웹 사이트 처리

K-WAH4.4은 보안이 강화된 웹 페이지의 주소 수집과 소스 수집이 가능하므로 더 많은 유형의 웹 사이트에 접근하여 접근성 평가가 가능합니다.

• 컴퓨터 리소스 제한 문제 개선

K-WAH4.4에서는 컴퓨터 리소스 부족 문제를 해결하였습니다. K-WAH4.4는 자동점검 페이지가 늘어남에 따라 컴퓨터 리소스를 많이 소모하여 웹주소 수집이 일정 범위를 넘어가면 프로그램이 다운되는 현상이 발생하였습니다. 그러나 K-WAH4.4에서는 웹주소 수집이 늘어나더라도 리소스의 소모가 늘어나지 않도록 개선하여 프로그램의 내구성을 향상시켰습니다.

• 저작권 표기

K-WAH4.31 이후 버전에서는 저작권 관련 사항을 강화하였습니다. 저작권 관련 사항은 K-WAH4.4 정보창의 저작권 링크 또는 저작권 보기을 참고하십시오. 또한 이 프로그램의 저작권은 한국정보화진흥원에 있으며, 웹와치(주)가 개발하고 버전 관리는 충북대학교 모비즈랩에서 담당하고 있습니다.
