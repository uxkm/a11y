단축키 목록


K-WAH 4.4는 키보드를 이용한 사용 방법을 지원함으로써 소프트웨어 접근성을 제공합니다.

  1. 메뉴 선택

    2. 모든 메뉴를 키보드로 수행할 수 있습니다.

  2. 결과 보기

    3. 결과 보기 화면에서 사용할 수 있는 단축키의 목록은 다음과 같습니다.

단축키

기능설명

Alt+U

점검한 URL 목록을 정렬방식에 따라 URL별 정렬과 지침별 정렬 방식으로 번갈아 전환하여 나타냅니다.

Alt+P

현재 프로젝트의 점검시 사용된 옵션 설정을 보여줍니다. 배치작업의 경우에는 배치설정을 보여줍니다.

Alt+T

오류 현황 표시 내용을 웹사이트의 전체의 오류 현황과 단일 웹페이지의 오류 현황으로 번갈아 전환하여 나타냅니다.

Alt+B

URL 목록에서 현재 선택된 웹페이지를 브라우저에 번갈아 나타내고 감춥니다.

Alt+S

URL 목록에서 현재 선택된 웹페이지 소스에서 수행할 수 있는 기능의 목록을 나타냅니다. 각 기능에 할당된 키를 누르면 기능을 수행할 수 있습니다. 찾기 창을 나타나게하는 것은 웹소스 창에 키보드 초점이 있을 때에 ' Ctrl+F' 키로도 실행이 가능합니다.

찾기(F)

Alt+F

또는

Ctrl+F

찾기 창이 열리고 찾기 기능을 실행할 수 있습니다. 찾기창의 옵션은 대문자 구분 여부를 설정할 수 있습니다. 사용방법은 찾기창의 '이전', '다음'버튼을 클릭하거나 Tab키로 초점을 이동한후 스페이스 바를 누르먄 됩니다.

키보드 초점이 텍스트 상자에 있을 때에 'Enter'키를 누르거나 'Shift+Enter"키를 이용해서도 다음 찾기와 이전 찾기가 수행됩니다.

'다음' 및 '이전' 버튼을 누르더라도 초점은 그대로 버튼에 남아 있습니다. 찾은 부분은 하이라이트되며, 소스코드 영역의 해당 위치로 초점을 이동하려면 Tab키를 눌러 소스코드 영역으로 초점을 이동하면 됩니다.

찾기 창을 화면에서 없애려면 '다음' 버튼을 누르거나 클릭합니다. 이 때 키보드 초점은 소스코드 창에 남아있게 됩니다.

전체선택(A)

소스코드 영역전체의 소스코드를 선택합니다. 이 때 소스코드 영역 전체가 하이라이트됩니다.

소스 복사(C)

선택된 영역의 소스코드를 클립보드로 복사합니다.

URL 복사(L)

선택된 URL을 클립보드로 복사합니다.단축키의 충돌을 피하기 위하여 'Alt+L"로 변경되었습니다.

※ 참고: 영역 간의 키보드 초점이동은 'TAB'키(정방향 이동)와 'Shift-TAB'키(역방향 이동)를 이용합니다. 자세한 사항은 키보드를 이용한 결과화면 탐색을 참고하십시오.
