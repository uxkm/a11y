점검 결과 보기


점검 완료 후 점검결과 보기

  • 점검결과 창은 3개의 컬럼으로 구성되어 있습니다.



    • ① URL 목록 : 웹접근성을 점검한 URL 목록을 보여줍니다.

    버튼 ⓐ를 이용하면 URL 목록의 정렬 방식을 선택할 수 있습니다.

    • [지침별로 정렬] : 오류가 발견된 웹페이지 목록을 세부검사항목 및 점검규칙별로 정렬하여 보여줍니다.
    • [URL별로 정렬] : 점검을 수행한 모든 URL의 목록을 점검한 순서대로 정렬하여 보여줍니다.

    ② 오류 현황 : 웹사이트(페이지)에서 발견된 웹접근성 오류 현황을 보여 줍니다.

    • 페이지별 오류 현황 : URL 목록에서 선택한 웹페이지에서 발견된 오류수를 세부지침별로 보여줍니다.

      • 버튼 ⓒ를 이용하면 전체 페이지의 오류수를 합산한 세부지침별 통계를 볼 수 있습니다.

    • 오류 목록 : URL 목록에서 선택한 웹페이지에서 발견된 오류의 목록을 보여줍니다.
      • 오류종류 : 오류의 종류
      • 라인 : 소스코드에서 오류가 발견된 해당 라인 번호
      • 열 : 소스코드에서 오류가 발견된 라인의 해당 열 번호

    ③ 소스 보기 : URL 목록에서 선택한 웹페이지의 소스코드를 보여줍니다.

    • 버튼 ⓓ를 이용하면 웹페이지를 보거나 감출수 있습니다.
    • 버튼 ⓔ를 이용하면 웹페이지 화면을 인쇄할 수 있습니다.
    • 버튼 ⓕ를 이용하면 소스코드의 표시형식을 바꾸거나 찾기 등의 기능을 이용할 수 있습니다.
    • 버튼 ⓖ를 이용하면 웹페이지 화면을 인쇄할 수 있습니다.
    • 버튼 ⓗ를 이용하면 현재 웹페이지 URL에 해당하는 웹페이지를 컴퓨터에 설치된 웹브라우저를 이용하여 렌터링 할 수 있습니다.

    ※ K-WAH4.4에서는 웹 브라우저 창에 접근성 진단시에 캡쳐한 페이지 이미지를 보여줍니다. 만일 해당 URL의 현재 웹 페이지 모습을 보려면 '팝업'버튼(ⓗ)을 누릅니다.

  • 오류목록에서 항목을 선택하면 오류가 발생한 소스코드 영역이 하이라이트 됩니다.


기존 프로젝트 점검결과 보기

  • 이전에 자동 점검을 수행한 프로젝트를 열어 점검결과를 재확인할 수 있습니다. 다음 중 한 가지 방법으로 점검결과를 열 수 있습니다.

    • 메뉴의 [파일]-[새로만들기]-[프로젝트]를 선택
    • 시작페이지의 최근점검한 프로젝트에서 [프로젝트]-[점검결과 열기]를 선택
    • Alt+F → O → P
  • [프로젝트 선택] 창이 열립니다.


  • 프로젝트 이름을 선택하고 [열기] 버튼을 누르면 점검결과 창이 열립니다.
